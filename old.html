Comparing Input Convex Networks to Feedforward Networks
S&DS 430 Optimization Techniques, 2019

I take a deep-dive into the intersection of optimization and deep learning to evaluate performance of input convex networks against standard feedforward networks.
Style Transformation on Human Faces Using Feedforward and Generative Methods
CPSC 459 Building Interactive Machines, 2019

We investigate multiple methods, including FastCNN, CycleGAN, and StyleGAN for style transformation (i.e. aging) on human faces.
Towards Understanding Learned GAN Distribution
CPSC 645 Topics in Theoretical Machine Learning, 2019

I studied learned distribution of 2-class convolutional GANs as the distributions of both classes varied and looked at latent disentanglement as a method to produce more monotonic results.
Teaching Baxter to Use Tools
CPSC 473 Intelligent Robotics Lab, 2018

We train the Baxter robot in Mujoco simulation using PPO to learn to use a tool to push an object across table and demonstrate sim2real transfer onto the physical robot.
